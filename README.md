# FindMe - Lost and Found System

FindMe is a web-based Lost and Found application developed to help users report, search, and manage lost and found items efficiently.

##  Project Overview

This project was developed collaboratively as part of a university software development project.

##  Features

- User Authentication
- Report Lost Items
- Report Found Items
- Search Lost & Found Items
- Update Item Status
- User Dashboard

## Tech Stack

- Python
- Flask
- PostgreSQL
- HTML
- CSS
- JavaScript
- Bootstrap

## Installation

1. Clone this repository

```bash
git clone https://github.com/destyaayud31/findme-lost-and-found.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Configure the database.

4. Run the application.

```bash
python app.py
```
